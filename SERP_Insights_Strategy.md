## 🎯 指標意義與利用策略

### 1. AISEO 觸發率 83.3% - 超高優先級優化

#### **意義**
20/24 查詢觸發 Google AI Overview，代表：
- ✅ 用戶期待「直接答案」而非長篇內容
- ⚠️ 自然排名點擊率會下降（AI Overview 佔據注意力）
- 🎯 **機會點**：成為 AI Overview 的引用來源（顯示來源連結）

#### **內容策略**

**文章結構調整：**
```
📄 必須包含的 4 個區塊（對應 settings.yaml）：

✅ Block 1: Quick Summary (100-150字)
   目的：被 AI Overview 引用
   格式：
   - 第一句：直接定義（「微量吸管是...」）
   - 第二句：核心用途（「主要用於...」）
   - 第三句：主要類型（「分為手動式與電動式」）

   範例：
   「微量吸管（Micropipette）是一種精密實驗工具，用於準確移取 0.1-10000μL
   的液體。主要應用於分子生物學、臨床診斷、生化檢測等領域。依據操作方式
   分為手動式與電動式兩大類。」

✅ Block 2: Definition (300-400字)
   目的：提供比 AI Overview 更深入的說明
   必須包含：
   - 基本定義（與其他工具的差異）
   - 技術規格（精度、容量範圍）
   - 應用場景（3-5 個具體案例）
   - 視覺化元素（圖表、流程圖）

✅ Block 3: Uses (500-600字)
   目的：回答所有 PAA 問題
   至少 3 個子章節：
   - 使用方法與技巧
   - 維護與清潔
   - 常見問題排查

✅ Block 4: FAQ (600-1000字)
   目的：覆蓋所有 PAA + Related Searches
   5-10 個問答對
```

**Schema.org 標記（必須）：**
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "微量吸管完整指南",
  "author": {...},
  "datePublished": "2025-11-20"
}

{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "微量吸管是什麼？",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "微量吸管是..."
      }
    }
  ]
}
```

### 2. 平均 3.2 個 PAA - 內容主題地圖

#### **意義**
Google 已經幫我們做好使用者研究：
- 這些是**真實用戶的搜尋意圖延伸**
- PAA 問題 = 高價值長尾關鍵字
- 競爭對手可能沒有完整回答這些問題

#### **從 SERP 數據提取的 PAA 問題**

**查詢：微量吸管**
```
1. 微量吸管是什麼？         → Block 1 (Quick Summary)
2. 微量吸管多久校正？       → Block 3 (維護章節)
3. 一般吸管是幾公分？       → Block 2 (規格說明)
4. Pipette 要按到底嗎？     → Block 3 (使用技巧)
```

#### **內容策略**

**1. 直接用於 FAQ 區塊**
- 每個 PAA 問題 = 1 個 H3 標題
- 提供 150-200 字完整答案
- 使用 FAQPage Schema 標記

**2. 發現內容缺口**
```
高優先級 PAA（競爭對手沒答好）：
✅ 「Pipette 要按到底嗎？」
   → 這是操作技巧的核心問題
   → 需要詳細圖解說明「第一停點」vs「第二停點」

✅ 「微量吸管多久校正？」
   → 實用維護問題
   → 需要提供具體時間表（每 X 個月或 X 次使用）
```

**3. 建立內部連結網絡**
```
主文章：微量吸管完整指南
  ↓
子文章 1：微量吸管校正完整流程（從 PAA「多久校正」延伸）
子文章 2：微量吸管使用技巧詳解（從 PAA「按到底」延伸）
子文章 3：微量吸管選購指南（從 Related Searches 延伸）
```

**策略建議：**
✅ 優先優化「最有機會被 AI 引用」的內容
